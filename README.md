# 🏡 RealtyApp

**Real Estate Social Media Platform** built like X (Twitter).

### Features
- Very lenient & fair algorithm for all users
- Beautiful property listings with photos, location, and price
- Verification badges (Blue check + Premium+)
- Premium plans: **$3/month** and **$10/month** with visibility boosts
- Monetization eligibility: 350 followers + 3.5M impressions

**Tech Stack**: Next.js 15 + Tailwind CSS + Prisma

Ready for Netlify deployment!
